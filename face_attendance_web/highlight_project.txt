🖼️ FACE RECOGNITION TECH
-----------------------
• Library: face_recognition (dlib-based)
• Algorithm: HOG + Linear SVM for face detection
• Encoding: 128-dimensional face embeddings
• Matching: Euclidean distance < 0.6 threshold
• Storage: Local folder structure per student
